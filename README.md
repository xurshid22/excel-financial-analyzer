# Финансовый анализатор на Streamlit

Веб-сервис для автоматического анализа себестоимости и продаж на основе Excel/CSV файлов.

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Запуск приложения

```bash
streamlit run app.py
```

Приложение откроется в браузере по адресу `http://localhost:8501`.

## 📂 Файлы для тестирования

В папке `sample_data/` находятся примеры файлов:
- `cost_sample.xlsx` — файл себестоимости
- `sales_sample.xlsx` — файл продаж

## 📋 Функциональность

1. **Автоматическое сопоставление колонок** — приложение распознаёт колонки по их названиям (Артикул, SKU, ID, Код и т.д.)
2. **Объединение данных** — по ключу ID/Артикул с агрегацией дубликатов
3. **Расчёт метрик** — общая себестоимость, валовая прибыль, маржинальность, ROI
4. **Классификация** — статусы товаров по маржинальности
5. **Визуализация** — интерактивные графики и KPI-карточки
6. **Экспорт** — скачивание результатов в форматированный Excel

## 📊 Расчётные метрики

| Метрика | Формула |
|---------|---------|
| Общая себестоимость | `Продано шт × Себестоимость ед.` |
| Валовая прибыль | `Выручка − Общая себестоимость` |
| Маржинальность | `(Прибыль / Выручка) × 100%` |
| ROI | `(Прибыль / Себестоимость) × 100%` |

## 🎨 Статусы товаров

- 🟢 **Высокомаржинальный** — > 30%
- 🟡 **Среднемаржинальный** — 10–30%
- 🟠 **Низкомаржинальный** — 0–10%
- 🔴 **Убыточный** — < 0%

## 🛠 Технологии

- **Streamlit** — веб-интерфейс
- **Pandas** — обработка данных
- **OpenPyXL** — форматированный Excel-экспорт
- **Plotly** — интерактивные графики (опционально)

## 📝 Примечания

- Приложение автоматически агрегирует дубликаты в файле продаж (суммирует количество и выручку)
- Если в файле себестоимости есть дубликаты по ID, оставляется первая запись
- Все числовые колонки автоматически очищаются от пробелов и запятых
- Регистр ID не учитывается (все приводятся к верхнему регистру)
